# Invoice MVP (Australia)

Simple invoice generator for Australian contractors and tradies.

## Features
- Generate PDF invoices
- Chat-based invoice creation (natural language)
- ABN compliant
- Optional GST support (planned)

## Tech Stack
- Next.js
- React
- pdf-lib
- OpenAI API

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

Create a `.env.local` file:

\`\`\`
OPENAI_API_KEY=your_key_here
\`\`\`

Open http://localhost:3000
